This project is made to help a business owner setup their invoices in a specific format in a relatively convinent way.

In the folder "flsk_lrn_three" there is a practice project that I have created to learn how to utilize and manage flask.
The project creator's code is at: https://github.com/jakerieger/FlaskIntroduction

In the folder "business_proj_one" exists the files of the project and the virtual environment where the files can run within.

In the folder "migrate_data_xl_sql" is code files and data that I created to get the data from excel file into a database for the business project.
